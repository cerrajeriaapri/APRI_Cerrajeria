# Abrir en Android Studio

Proyecto listo para abrir:

`C:\Users\cerra\Documents\Codex\2026-05-23\files-mentioned-by-the-user-whatsapp\android-app`

## Pasos

1. Abrir Android Studio.
2. Tocar `Open`.
3. Seleccionar la carpeta `android-app`.
4. Esperar a que termine `Gradle Sync`.
5. Para probar: tocar el boton verde `Run`.
6. Para Play Store: `Build > Generate Signed App Bundle / APK`.

## Si Android Studio pregunta

- Confiar/Trust project: `Trust Project`.
- Descargar Gradle/Android Gradle Plugin: aceptar.
- Instalar SDK faltante: aceptar.

## Archivo para Play Store

El archivo final que se sube a Play Store es un `.aab`, generado desde:

`Build > Generate Signed App Bundle / APK > Android App Bundle`
